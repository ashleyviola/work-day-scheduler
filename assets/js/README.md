# Work Day Scheduler - Homework # 5
## Project Description 
Website designed to help organize activities during the work day. Scheduler will turn grey when time has past, highlight the current hour red and the future will be green. 

### Built With 
- HTML
- CSS 
- Javascript
- JQuery 

### Files & Directories 
- index.html
- style.css
- script.js
